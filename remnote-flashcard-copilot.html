RemNote Flashcard Copilot

A flashcard writing copilot integrated into a note-taking application, designed to enhance the educational experience for students. With inline suggestions and a grammarly-style correctness check, this copilot ensures the production of high-quality flashcards, aiding over 700k users in their learning journeys.

The tool not only eases onboarding for new users by providing expert feedback, but it also serves as an educational scaffold for more seasoned students, encouraging self-improvement in flashcard creation rather than dependence on AI-generated content.

Addressing Challenges:

  • Minimizing Latency: Leveraging gpt-3.5-turbo-instruct for rapid response times, critical for maintaining user engagement.
  • User Interface Design: Creating an intuitive and aesthetically pleasing interface.
  • Prompt Engineering: Crafting prompts that cater to a global user base, accommodating multiple languages.
