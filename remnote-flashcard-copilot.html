RemNote Flashcard Copilot

A flashcard writing copilot integrated into a note-taking application with over 700k registered users.

The tool not only eases onboarding for new users by providing expert feedback, but it also serves as an educational scaffold for more seasoned students, encouraging self-improvement in flashcard creation rather than dependence on AI-generated content.

Main Challenges:

  • Backend: I built RemNote's LLM request backend used across all RemNote's AI projects.
  • Minimizing Latency: I implemented partial JSON parsing to stream GPT function call results to the user as they are generated.
  • Prompt Engineering: Crafting prompts that cater to a global user base, accommodating multiple languages.
  • Reliability: I built extensive prompt evaluation suites to ensure high quality completions.
