TypeScript Type Fixing Agent

An open source AI agent which automates fixing type errors in large 100,000+ line TypeScript code bases.

Types are like labels that describe what kind of data you can work with in a programming language and type errors are like mislabelled data. Type errors often indicate the existence of a bug. Type errors are easy to identify, but in large code bases they often require complex chains of reasoning to solve because they can span many different files and require a lot of effort to fix.

My type fixing agent solves this problem by autonomously grinding away fixing type errors while you sleep. When you wake up there will be a nice stack of well-formatted PRs for you to review and either merge or fix.

This project required learning the compiler API for the TypeScript programming language. The compiler API was incredibly useful because it allowed me to provide precise context to the LLM agent to improve multi-step reasoning performance.
